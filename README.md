# AtmoSync: Micro-Climate Arbitrage Analytics

## Project Overview

AtmoSync is a Data Engineering project developed as part of the Infotact Solutions Internship. The project monitors environmental conditions inside agricultural shipping containers using simulated IoT sensor data. It helps identify spoilage risks and supports better transportation decisions by providing real-time insights and alerts.

## Problem Statement

Traditional supply chain systems rely on estimated transit times and general weather forecasts. They cannot monitor real-time micro-climate conditions inside shipping containers. Sudden changes in temperature, humidity, or vibration can cause fruits and vegetables to spoil before reaching their destination, resulting in financial losses.

---

## Project Objective

The objective of this project is to:

- Simulate IoT sensor data for shipping containers.
- Monitor temperature, humidity, and vibration.
- Detect spoilage risks.
- Recommend shipment rerouting based on environmental conditions.
- Build a real-time Data Engineering pipeline and dashboard.
